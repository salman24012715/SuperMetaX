## Overview

Super-Meta-X (SMX) Enterprise is an advanced, automated statistical environment
for comprehensive meta-analysis and evidence synthesis, built on R's `meta` and
`metafor` ecosystems.

## Documentation

Open **[SMX_engine.html](SMX_engine.html)** for the full documentation with a
17-figure visual reference gallery.
